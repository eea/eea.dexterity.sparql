==========
EEA Dexterity Sparql
==========
EEA Dexterity Sparql is a plone product for fetching data from Linked open data servers
(sparql endpoints).


.. contents::


Installation
============
  * Go to admin > Site Setup > Add-ons
  * Activate EEA Dexterity Sparql


Usage
=====

* Within your working Folder add a Sparql or a Sparql Bookmarks Folder
